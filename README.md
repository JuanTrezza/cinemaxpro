## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Sobre el Proyecto

CINEMAX PRO es una plataforma premium de descubrimiento de películas inspirada en Netflix. Consume la API de The Movie Database (TMDb) en tiempo real para mostrar películas en tendencia, mejor calificadas, populares y próximos estrenos.

Modelo de negocio: B2C — dirigido al consumidor final que busca explorar y descubrir contenido cinematográfico.

## Tecnologías Utilizadas

| Tecnología | Descripción |
|---|---|
| Google Stitch | Prototipado y construcción asistida de interfaz |
| Google AI Studio | Base inicial y soporte de generación |
| React 19 + TypeScript | Desarrollo frontend moderno, tipado y escalable |
| Tailwind CSS v3 | Sistema de estilos utility-first y diseño responsive |
| Framer Motion | Animaciones fluidas e interacciones visuales |
| Vite 6 | Bundling y desarrollo ultra-rápido |
| TMDb API | Fuente de datos cinematográficos en tiempo real |
| GitHub Actions | Automatización de build y despliegue |
| GitHub Pages | Hosting estático del proyecto |

## Características

- 🎥 Hero cinematográfico con auto-rotación de películas en tendencia
- 🔎 Búsqueda en tiempo real con debounce
- 🧩 Filtros por género funcionales
- ▶️ Modal de detalle con trailer de YouTube integrado
- ❤️ Sistema de favoritos con persistencia en localStorage
- 🍿 Sección "Continuar viendo" personalizada
- 📊 Estadísticas personales del usuario
- 🔴 Estilo Netflix con rojo signature #E50914
- ✨ Animaciones suaves con Framer Motion
- 📱 Diseño responsive mobile-first

## API

Integración con The Movie Database (TMDb):

- Endpoints: trending, top_rated, popular, upcoming, search
- Idioma: español Argentina (es-AR)
- Imágenes en alta resolución
- Información de cast, director y trailers
